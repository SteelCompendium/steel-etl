## Features — by level (90 terms)

These are class and circle features (non-ability). Where a feature is also an ability entry point (e.g. "Call Forth" at level 1 appears both as a feature wrapper and as an ability), the feature code targets the feature wrapper section; the ability code (`feature.ability.*`) targets the stat-block entry. Prefer the ability code when the prose clearly refers to the activated game ability; prefer the feature code when referring to the feature itself (e.g. "the Call Forth feature").

## Disambiguation — common-word ability/feature names (REQUIRED)

The Summoner book uses many short, common-word names for its abilities and features: **Essence**, **Minions**, **Shield!**, **Halt!**, **Rise!**, **Formation**, **Portfolio**. Most of these words also appear throughout the text as ordinary English. The same rules that govern the Heroes book apply here — link ONLY the game-mechanic reference, never ordinary prose, and never inside the defining section itself (self-ref rule).

**General rule for Summoner mechanics:** If you could substitute "the [X] feature" or "the [X] ability" and the sentence still makes mechanical sense, it is a game-mechanic reference. When in doubt, skip the link.

**Quick-command abilities (Focus Fire!, Halt!, Not Yet!, Shield!):** these are commands issued to minions. The exclamation mark in the heading text is part of the name. In prose, the word often appears without the exclamation mark ("the summoner calls halt", "focuses fire"). Link only when the passage clearly describes activating or referencing the ability as a mechanic, not when it describes the flavor action.

**Feature vs. ability codes for shared names:** Call Forth, Strike for Me, Minion Bridge, Summoner Strike, Focus Fire!, Halt!, Not Yet!, and Shield! each have a `feature.summoner.*` code (the feature wrapper) AND a `feature.ability.*` code (the stat-block entry). Use the **ability code** when the prose describes activating the mechanic or its cost/effect. Use the **feature code** only when referring to the feature section itself (e.g. "see the Call Forth feature").
